# LINX株式会社 コーポレートサイト

## 概要
群馬県藤岡市を拠点とするLINX株式会社のコーポレートサイトです。

## 会社情報
- **会社名**: LINX株式会社
- **所在地**: 群馬県藤岡市藤岡803-2
- **代表取締役**: 西村 涼太郎
- **事業内容**: アウトソーシング業

## デザイン仕様
- **メインカラー**: 黄色（#FFD600）/ 水色（#00BCD4）
- **アクセント**: グラデーション（黄→水色）
- **背景**: 白・ライトグレー・ダーク（セクションで使い分け）
- **ドット柄**: 各セクション背景に使用

## ページ構成
| セクション | 内容 |
|-----------|------|
| ヒーロー | 白背景・黒文字・右側にサービスカード |
| 実績バー | 取引企業数・満足度・プロジェクト数・サポート体制 |
| ABOUT US | 代表メッセージ + 会社概要テーブル |
| 事業内容 | BPO / 人材派遣 / コンサルティング |
| 選ばれる理由 | 3つの強み |
| CTAバナー | お問い合わせ誘導 |
| お問い合わせ | バリデーション付きフォーム |

## ファイル構成
```
linx/
  index.html        メインページ
  css/style.css     スタイルシート
  js/main.js        JavaScript（アニメーション・フォーム・カウントアップ）
```

## 機能
- フォームバリデーション
- スクロールアニメーション（fade-up）
- カウントアップアニメーション（実績数値）
- ハンバーガーメニュー（スマートフォン対応）
- トップへ戻るボタン
- レスポンシブデザイン
