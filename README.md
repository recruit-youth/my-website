# トップエージェント株式会社 コーポレートサイト

## 概要
群馬県藤岡市を拠点とするトップエージェント株式会社の採用・会社紹介コーポレートサイトです。

## 会社情報
- **会社名**: トップエージェント株式会社
- **所在地**: 群馬県藤岡市中987-1
- **代表取締役**: 高橋 将汰
- **事業内容**: 保険代理業務 / 有償車貸渡事業 / 新車・中古車販売

## デザイン仕様
- **メインカラー**: 赤 (#CC0000)
- **サブカラー**: 白・黒
- **アクセント**: ドット柄（各セクションの背景に使用）
- **トップ画面**: 白背景・黒文字（修正済み）
